# 📱 Diagramas de Flujo de Aplicaciones de Mensajería

## 🎯 Descripción del Proyecto

Este proyecto contiene diagramas de flujo completos de aplicaciones de mensajería populares, diseñados para análisis de UX/UI y optimización de contenido. Cada diagrama muestra la navegación completa entre pantallas con imágenes reales y diseño profesional estilo Figma.
